# sql-data-analytics-project

A structured set of SQL scripts built for exploring, analyzing, and reporting on relational data. The project walks through core analytical patterns that data analysts and BI professionals use daily — from basic database exploration to advanced trend and segmentation analysis.

This repository serves as both a learning resource and a practical toolkit: each script is self-contained, focused on one analytical concept, and written to be easily adapted to your own datasets. Whether you're investigating how metrics evolve over time, calculating running totals, breaking down data by category, or comparing parts to a whole, you'll find a ready-to-use query here.
## What's covered

- Database exploration — understanding table structures, relationships, and data types
- Dimensions exploration — exploring unique values and categories within key dimensions
- Date range exploration — identifying the time span and boundaries of the data
- Measures exploration — calculating core business metrics and totals
- Magnitude analysis — comparing values across categories to understand scale
- Ranking analysis — identifying top and bottom performers
- Change over time analysis — analyzing how values change over days, months, or years
- Cumulative analysis — running totals and moving calculations
- Performance analysis — comparing current performance against targets or previous periods
- Data segmentation — grouping and comparing data across categories
- Part-to-whole analysis — understanding proportions within a dataset
- Customer report — a consolidated report analyzing customer behavior and value
- Product report — a consolidated report analyzing product performance
